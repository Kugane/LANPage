# LANPage
A simple website template for your LAN Party that integrates with LAN Launcher

## Things required:
* Webserver and PHP5 or newer
* PHP SQLite extension enabled

## Installation:
- Copy or rename config.sample.php to config.php and change values
- chmod 0755 db/*.db
